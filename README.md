# Electronic Health Record (EHR) Database - SQL Project

## 📌 Project Overview

This project simulates an Electronic Health Record (EHR) system used in healthcare organizations to manage patient records, doctor information, appointments, medical records, and billing data.

## 🛠️ Tools Used

* SQL Server
* SQL
* GitHub

## 🗄️ Database Tables

* Patients (75 records)
* Doctors (10 records)
* Appointments (50 records)
* MedicalRecords (50 records)
* Billing (60 records)

## 📊 Key SQL Queries

* Most visited doctor
* Busiest day
* Most common diagnosis
* Revenue by doctor
* Monthly revenue
* Patients per doctor

## 🎯 Skills Demonstrated

* Database Design
* SQL Joins
* Aggregations
* Data Analysis
* Healthcare Data Management

## 📁 Project Files

* schema.sql (Database structure)
* insert_data.sql (Sample data)
* ehr_queries.sql (Analysis queries)

## 🚀 Outcome

Built a healthcare database system to manage clinical and financial data efficiently using SQL Server.
